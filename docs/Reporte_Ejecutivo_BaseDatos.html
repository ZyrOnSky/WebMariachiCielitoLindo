1. Resumen Ejecutivo

El presente documento detalla el proceso técnico integral llevado a cabo por ABVC Digital Constructions para la actualización del catálogo digital del Mariachi Cielito Lindo.

Se modernizó la gestión del repertorio integrando la base de datos Firestore y un registro ampliado provisto en formato Excel.

El Desafío Principal

Sincronizar una base de datos activa con 255 canciones base con un nuevo listado de 406 títulos en Excel, eliminando duplicados y garantizando la presencia de videos musicales (YouTube).

2. Hitos Técnicos y Metodología

1

Auditoría y Extracción

Se extrajo la información de Firestore y se analizaron las tablas de Excel para detectar registros superpuestos con diferencias ortográficas.

2

Estandarización de Datos

Proceso de limpieza avanzado para combinar ambas listas sin duplicados, corrigiendo errores comunes y conservando la mejor calidad multimedia.

3

Enriquecimiento de Categorías

Se expandió el etiquetado. A las canciones con una sola categoría se les añadieron múltiples ocasiones y géneros relacionados (Ej: Serenata -> Aniversarios), optimizando las búsquedas.
4

Multimedia e Interactividad

Vinculación de más de 200 nuevos enlaces de YouTube. El sistema ahora permite previsualizar audio directamente en el panel administrativo.
