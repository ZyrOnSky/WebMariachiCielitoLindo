Antecedentes del Proyecto

El proyecto inició ante la necesidad crítica de reemplazar la antigua solución basada en WordPress, la cual presentaba una experiencia de usuario deficiente, tiempos de carga lentos y un catálogo de canciones difícil de navegar.

La solución ejecutada consistió en el desarrollo de una Arquitectura desde Cero, priorizando la velocidad, la seguridad y una interfaz elegante que permitiera a Mariachi Cielito Lindo destacar profesionalmente en el mercado de Guayaquil.

Análisis de Tráfico y Costos de Operación

La web utiliza tecnología de punta en la nube a través de Netlify. Es importante entender cómo funciona el consumo de datos para mantener la gratuidad del servicio el mayor tiempo posible.

1. Cuota Gratuita Mensual

Netlify ofrece 100 GB de transferencia de datos gratuita cada mes. Dado que la web pesa 15 MB por carga, usted tiene un margen aproximado de:

~ 6,600 visitas gratis / mes

(Estimación conservadora para asegurar que nunca exceda el límite en uso normal)

2. Manejo de Excedentes y Campañas

Si Mariachi Cielito Lindo realiza una campaña masiva (ej. Día de las Madres) y supera las 6,600 visitas, Netlify aplicará un cargo de:

$ 10.00 USD por paquete adicional

Este paquete le otorga un "boost" de tráfico por 30 días.
Estrategia ABVC: Milton monitoreará las tasas de consumo y le notificará con antelación para activar esta mejora solo si es estrictamente necesario, protegiendo su presupuesto.
